Question-1: What are the different ways to select an element in the DOM?

There are several ways to select elements from the DOM:We can select a specific element by ID,multiple elements by class,all elements of a certain tag by tag name,the first matching element with querySelector() and all matching elements with querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML returns everything including HTML content,innerText gives only the visible text and textContent returns all the text,including hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is when I add an event listener to a parent element,which handles events for its child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts from a child element and then bubbles up to its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript,I use createElement() to create a new element and appendChild() to add it to a parent.To remove an element,I use removeChild().
